/v1/players?search=[term] – GET


Returns a list of players that contain the search term in their email, first, last or nickname

Successful response

HTTP Status code - 200

          
          [
            {
              "id": 1,
              "email": "player_1@email.com",
              "first_name": "First",
              "last_name": "Last",
              "nickname": "Nickname 1"
            }
          ]

When no results are found an empty list is returned
