Logging in a player

Return

/v1/sign_in – POST


          
            {
              "player": {
                "email": "test@example.com",
                "password": "[password]"
              }
            }

Successful response

HTTP Status code - 200

          
            {
              "player": {
                "email": "test@example.com",
                "first_name": "First",
                "last_name": "Last",
                "nickname": "Nickname",
                "auth_token": "[authentication_token]"
              }
            }

Error response - Missing required parameters

HTTP Status code - 422

          
            {
              "errors": [
                "Missing required parameters"
              ]
            }

Error response - Incorrect email/password combination

HTTP Status code - 401

          
            {
              "errors": [
                "The email and or password are not valid"
              ]
            }
